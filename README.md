# Royal Weaves - Premium Ethnic Shop

A lightweight, high-performance e-commerce frontend for luxury ethnic wear.

## Technology Stack
- **Backend**: FastAPI (Python)
- **Frontend**: Vanilla HTML5, CSS3, and JavaScript
- **Templates**: Jinja2

## How to Run locally

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the Server**:
   ```bash
   python main.py
   ```

3. **Access the App**:
   Open [http://localhost:8000](http://localhost:8000) in your browser.

## Features
- Fully responsive premium design.
- Lightning-fast page transitions (Vanilla JS).
- Lightweight footprint (No node_modules).
- Easy-to-manage JSON-based product storage.
